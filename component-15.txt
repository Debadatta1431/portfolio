
## Troubleshooting

- **Error: Cannot find module '@/components/ui/button'**
  - Make sure you ran `npx shadcn@latest add button card` successfully.
  - Verify that `src/components/ui/button.tsx` and `src/components/ui/card.tsx` exist.

- **Error: Cannot find module '@/lib/utils'**
  - Ensure you created `src/lib/utils.ts` with the `cn` function as shown in Step 5.

- **Tailwind styles not applying**
  - Check that `tailwind.config.js` has the correct `content` paths.
  - Ensure `src/index.css` has the three `@tailwind` directives.

- **TypeScript path alias errors**
  - Verify `vite.config.ts` and `tsconfig.json` both have the `@/*` path alias configured correctly.
  - Restart the dev server after making changes to config files.
